DuoPlayer 개인정보 처리방침

최종 업데이트: 2024년 12월 | 시행일: 2024년 12월

🔒 개인정보 보호 약속

DuoPlayer는 완전히 로컬화된 비디오 플레이어 애플리케이션입니다. 저희는 귀하의 개인 데이터를 외부 서버에 수집, 저장 또는 전송하지 않습니다. 모든 데이터 처리는 귀하의 기기에서 수행되어 개인정보 보호와 데이터 보안을 보장합니다.

애플리케이션 개요

DuoPlayer는 다양한 비디오 형식, 자막 표시, 음성 전사 및 다국어 번역 기능을 지원하는 기능이 풍부한 iOS 비디오 플레이어입니다. 본 개인정보 처리방침은 이 애플리케이션을 사용할 때 귀하의 데이터와 개인정보를 어떻게 처리하는지 자세히 설명합니다.

데이터 처리 방식

📱 로컬 데이터 처리

이 애플리케이션은 완전히 로컬화된 처리 설계 철학을 채택하여 모든 기능이 귀하의 기기에서 실행됩니다:

  • 비디오 재생: 기기에서 비디오 파일을 직접 재생
  • 음성 전사: 기기에서 로컬로 음성을 텍스트로 변환
  • 텍스트 번역: 기기에서 로컬로 텍스트 번역 수행
  • 데이터 저장: 모든 사용자 데이터는 귀하의 기기에만 로컬로 저장
⚠️ 중요 참고사항: 이 애플리케이션은 귀하의 데이터를 인터넷이나 제3자 서버로 전송하지 않습니다

로컬에 저장되는 데이터 유형

🎬 재생 기록

  • 저장 내용: 비디오 파일 경로, 재생 진행률, 마지막 재생 시간, 비디오 길이
  • 목적: 재생 진행률 복원, 편리한 기록 액세스 제공
  • 저장 위치: 기기 로컬 저장소
  • 접근 권한: 이 애플리케이션에서만 접근 가능

📝 자막 및 전사 데이터

  • 저장 내용: 생성된 자막 파일, 전사 텍스트, 번역 결과
  • 목적: 사용자 생성 자막 저장, 재시작 기능 지원
  • 처리 방식: 기기에서 완전히 로컬로 처리, 음성이나 텍스트 데이터 업로드 없음
  • 데이터 보안: 시스템 수준 보안 보호 메커니즘

⚙️ 애플리케이션 설정

  • 저장 내용: 언어 설정, 자막 스타일 설정, 플레이어 구성
  • 목적: 개인화된 사용자 경험
  • 저장 위치: 기기 로컬 저장소

💳 구독 정보

  • 처리 방식: Apple App Store 공식 서비스를 통해 처리
  • 데이터 관리: Apple에서 직접 관리, 결제 정보 저장하지 않음
  • 개인정보 보호: Apple의 개인정보 보호 및 보안 표준 준수

권한 사용

🎤 음성 인식 권한

권한 사용: 비디오를 위한 실시간 자막 및 전사 텍스트 생성

처리 방식: 내장된 시스템 음성 인식 사용, 기기에서 완전히 로컬로 처리

데이터 보안: 음성 데이터가 기기를 떠나거나 어떤 서버로도 전송되지 않음

제어: 시스템 설정에서 언제든지 이 권한을 취소할 수 있음

📁 파일 접근 권한

  • 접근 범위: 적극적으로 선택한 비디오 및 자막 파일로 제한
  • 보안 메커니즘: 시스템 수준 파일 접근 보호로 제어된 권한 보장
  • 데이터 보호: 명시적으로 선택하지 않은 파일에는 접근하지 않음

수집하지 않는 데이터

🚫 다음 데이터는 명시적으로 수집하지 않습니다:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 기기 식별자 또는 추적 정보
  • 위치 정보
  • 비디오 파일 콘텐츠
  • 음성 녹음 또는 오디오 데이터
  • 브라우징 기록 또는 사용 분석
  • 연락처 또는 주소록 정보
  • 사진 또는 앨범 콘텐츠 (비디오 파일을 적극적으로 선택하지 않는 한)

제3자 서비스

🍎 Apple 시스템 서비스

이 애플리케이션은 다음 내장된 Apple iOS 시스템 기능만 사용합니다:

  • 인앱 구매: App Store를 통한 구독 서비스 처리
  • 음성 인식: 내장된 시스템 음성-텍스트 변환 기능 사용
  • 텍스트 번역: 내장된 시스템 번역 서비스 사용
  • 비디오 재생: 내장된 시스템 미디어 재생 기능 사용
📍 제3자 분석, 광고 또는 데이터 수집 서비스는 사용하지 않습니다

데이터 보안 및 보호

🛡️ 보안 조치

  • 애플리케이션 격리: 애플리케이션 데이터는 시스템 보안 메커니즘으로 보호되어 다른 애플리케이션에서 접근 불가
  • 로컬 암호화: 민감한 데이터는 시스템 수준 암호화로 저장
  • 최소 권한: 기능에 필요한 최소 권한만 요청
  • 네트워크 전송 없음: 애플리케이션 데이터는 네트워크를 통해 전송되지 않음

🔒 데이터 보존

  • 재생 기록: 사용자가 수동으로 삭제할 때까지 보존
  • 자막 데이터: 사용자가 수동으로 삭제할 때까지 보존
  • 애플리케이션 설정: 애플리케이션 제거 시까지 보존
  • 완전 삭제: 애플리케이션 제거 시 모든 로컬 데이터 완전 삭제

아동 개인정보 보호

저희 애플리케이션은 모든 연령대에 적합합니다. 개인정보를 수집하지 않으므로 13세 미만 아동이 이 애플리케이션을 사용하는 것이 안전합니다. 부모님께서는 아동이 적절한 비디오 콘텐츠를 선택하도록 기기 사용을 감독하시기를 권장합니다.

귀하의 권리와 통제

✅ 귀하는 완전한 통제권을 갖습니다:

  • 데이터 접근: 모든 데이터가 귀하의 기기에 있어 언제든지 확인 가능
  • 데이터 삭제: 재생 기록 삭제, 자막 파일 삭제 또는 애플리케이션 제거 가능
  • 권한 관리: 시스템 설정에서 애플리케이션 권한 관리 가능
  • 기능 제어: 음성 전사 등의 기능 사용 여부 선택 가능

규정 준수 선언

📋 법적 준수

  • GDPR 준수: 개인 데이터를 수집하지 않아 자연스럽게 GDPR 요구사항 준수
  • CCPA 준수: 개인정보 판매나 공유와 관련 없음
  • COPPA 준수: 13세 미만 아동의 개인정보 수집하지 않음
  • Apple 정책: Apple App Store 심사 가이드라인 엄격 준수

개인정보 처리방침 변경

기능 업데이트나 법적 요구사항에 따라 이 개인정보 처리방침을 업데이트할 수 있습니다. 중요한 변경사항은 앱 내 알림을 통해 전달됩니다. 저희가 귀하의 개인정보를 어떻게 보호하는지 이해하기 위해 이 정책을 정기적으로 검토하시기 바랍니다.
